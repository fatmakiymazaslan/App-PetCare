# 🐾 PetCare - Ultimate Pet Management App

PetCare is a modern, responsive web application designed for pet owners to seamlessly manage their furry friends' daily routines, health, and needs.


## ✨ Key Features
* 👤 **User Management:** Secure Sign-up and Login system.
* 🐶 **Pet Profiles:** Add multiple pets with photos, nicknames, and unique profiles.
* ⚙️ **Smart Settings:** Update username and password with built-in **Data Migration** (your pets follow you even if you change your name!).
* 📝 **Pet Notes:** Keep track of medical history, allergies, or special instructions per pet.
* 🍖 **Nutrition Tracking:** Log food and water intake daily.
* 📅 **Calendar & Reminders:** Never miss a vaccine or a walk with the integrated reminder system.

## 🛠️ Tech Stack
* **HTML5 & CSS3:** Modern, responsive design with a focus on mobile-first UI.
* **JavaScript (Vanilla JS):** Advanced logic for state management and DOM manipulation.
* **LocalStorage:** Secure and persistent client-side data storage (No database setup required!).

## 🚀 How to Use
1. Clone this repository to your local machine.
2. Open `index.html` in your favorite web browser.
3. Start managing your pets like a pro!
